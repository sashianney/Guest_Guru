Welcome To Admin Panel...

In this panel, Admin can control several things. You can see in the top left corner, we have created a side nav-bar so that the admin can control all these functions. This includes the ability to view and update all the registered customer's details, view and update all the registered Hotel Admin's details. Additionally, Admin has the authority to approve pending hotel statuses. The Admin can also add hotel rooms, Categories & Services. We have created only a single Admin, so based on this, the Admin has the login and logout features only...
